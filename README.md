## 📌 About the Project

Most student projects are static pages with no real backend. NexaCon 2026 was built to go beyond that — simulating how a real production web application is structured and deployed.

It includes a multi-page frontend, a live REST API, a cloud database, a real payment gateway integration with cryptographic verification, and a decoupled deployment across three separate platforms.

---

## ✨ Features

- 5-page responsive frontend with light/dark theme toggle
- Animated stat counters, filterable speaker grid, 3-day schedule tab switcher
- Razorpay payment integration with HMAC-SHA256 signature verification
- Node.js + Express REST API deployed on Render
- MongoDB Atlas cloud database storing real registration data
- EmailJS contact form with live email delivery
